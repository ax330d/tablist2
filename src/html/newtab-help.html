TabList Help

What TabList Does

  • Display All Tabs:
    Shows every tab (title, last accessed, url, favicon) in your current window and updates as you add or remove tabs. Discarded tabs are semi-transparent.
  • Manage Tabs:
    Close tabs close, discard memory memory, or reload tabs refresh.
  • Switch Tabs:
    Click a tab's favicon to jump to it instantly.
  • Drag-and-Drop:
    Move tabs by dragging drag_indicator to reorder them or drop into groups for organization. Drag groups by holding header of the group.
  • Tab Groups:
    View and collapse groups; use Shift + click to select tabs, then create or move to groups.
  • Indicators:
    Marks no_encryption insecure, music_note audible, volume_off muted, ac_unit frozen, or keeppinned tabs for quick status checks.
  • Customize:
    Adjust theme and font sizes via the options page.
  • Keyboard controls:
    TabList is fully controllable by keyboard. For help on keyboard control see "Shortcuts" footer link.
