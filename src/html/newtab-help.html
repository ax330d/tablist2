TabList Help

What TabList Does

  • Display All Tabs:
    Displays every tab (title, last accessed time, URL, and favicon) in your current window and updates as you add or remove them. Discarded tabs are semi-transparent.
  • Manage Tabs:
    Close tabs close, discard memory memory, or reload tabs refresh.
    Use Shift + click to select tabs or groups (click on header), release Shift, then create or move to groups or windows.
  • Switch Tabs:
    Click a tab's favicon to jump to it instantly.
  • Drag-and-Drop:
    Move tabs by dragging drag_indicator to reorder them or drop into groups for organization. Drag groups by their headers.
  • Tab Groups:
    Click on tab header to collapse or expand group.
  • Indicators:
    Marks no_encryption insecure, music_note audible, volume_off muted, ac_unit frozen, or keeppinned tabs for quick status checks.
  • Customize:
    Adjust theme and font sizes via the options page.
  • Keyboard controls:
    TabList is fully controllable by keyboard. For help with keyboard controls, see the 'Shortcuts' link in the footer.
